# Webex API Lab

Deze directory bevat Python-scripts om de Webex REST API te testen.

## Scripts

- `authentication.py`  
  Test het personal access token met de `people/me` endpoint.

- `list-people.py`  
  Haalt gebruikers op op basis van e-mailadres.

- `list-rooms.py`  
  Toont alle rooms waar de geauthenticeerde gebruiker lid van is.

- `create-rooms.py`  
  Maakt een nieuwe Webex room aan.

- `get-room-details.py`  
  Vraagt extra informatie op over een room (meetingInfo indien beschikbaar).

- `list-memberships.py`  
  Toont de leden van een bepaalde room.

- `create-membership.py`  
  Voegt een gebruiker toe aan een room op basis van e-mailadres.

- `create-markdown-message.py`  
  Stuurt een bericht in Markdown naar een room.
